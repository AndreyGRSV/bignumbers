Detailed Description

template<const int digits = 100, const int prec = 0, class T = unsigned char>
class sag::bdig< digits, prec, T >

This template class cover manipulation with big numbers. Number size up to indicated in parameters.

Template Parameters
TType of element in array to hold number. This type can be unsigned integer type from char to long long and specific to compiler int128_t
digitsNumber of digits in integer part. Depends on system resources and can be as big as it required. Tested values up to 20'000
precNumber of digits after point.
